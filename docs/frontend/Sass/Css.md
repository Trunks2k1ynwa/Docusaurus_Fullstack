## Css selector
- Element selector
```css
p{
  color:blue;
}
```
- Class Selector
```css
.highlight {
  background-color: yellow;
}
```
- Id Selector
```css
#header {
  font-size: 24px;
}
```
- Attribute Selector : Selector theo thuộc tính
```css
input[type="text"] {
  border: 1px solid gray;
}
```
- Descendant Selector : Selector kết hợp
```css
li a {
  text-decoration: none;
}
```
- Độ ưu tiên:
ID, class,Attribute, element selectors
